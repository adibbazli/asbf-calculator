# ASBF Calculator

This calculator is to make direct comparison between ASBF and ASB Cash assuming same monthly payment

## View App

https://adibbazli.github.io/asbf-calculator/

## Run Locally

**Prerequisites:**  Node.js

1. Install dependencies:
   `npm install`
2. Run the app:
   `npm run dev`
3. Deploy the app:
   `npm run deploy`
